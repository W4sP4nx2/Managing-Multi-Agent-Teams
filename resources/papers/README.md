# 📄 Papers

Prefer linking to canonical sources such as arXiv, IEEE, ACM, official project pages, or publisher landing pages.

## Rules

- Do not commit large PDFs by default.
- Use Git LFS when a local PDF is required.
- Keep paper notes in Markdown when possible.
- Link each paper from [`../README.md`](../README.md) so learners can find it from the master index.

## Local PDF Exception

Only place a PDF here when the canonical source is unstable, inaccessible to learners, or required for offline classroom delivery.
